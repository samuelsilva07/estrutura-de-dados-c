# Estrutura de Dados em C
## Seja bem vindo! 💡
Este repositório contém todo os códigos que produzi durante meus estudos, baseado nas aulas ministradas da disciplina Estrutura de Dados I, do curso de Ciência da Computação da <abbr title="Universidade Federal Rural do Rio de Janeiro" style="text-decoration:none;">**UFRRJ**</abbr> 📕

Durante esse período, tive contato com vários algoritmos e conceitos fundamentais do tópico, entre eles:

- Algoritmos de ordenação
- Listas encadeadas
- Pilhas
- Filas
- Árvores

Além dos algoritmos e bibliotecas, alguns tópicos possuem **exemplos práticos**, em que pude aplicar os conceitos vistos em aula em situações mais reais e criativas 🔧

## 🔎 Observações

- Todos os algoritmos possuem comentários, que me ajudaram a internalizar e a entender melhor o funcionamento dos códigos durante meu aprendizado.<br> Espero que eles possam te ajudar também! :)
- Alguns tópicos também possuem implementações em Python 🐍
- Esse repositório **continuará em atualização**, pois estarei sempre buscando novas aplicações para internalizar e praticar os conteúdos aprendidos 🚀


## 💻 Como executar os códigos de exemplo?

1 - **Copie o repositório na sua máquina**

2 - Abra o **terminal** (bash) na pasta em que o código desejado está

3 - **Compile o código**, utilizando o comando abaixo:

```bash
gcc nome_do_arquivo.c -o nome_do_arquivo
```

4 - **Execute o programa**, digitando:

```bash
./"nome_do_arquivo"
```

E pronto! O código será executado perfeitamente.
